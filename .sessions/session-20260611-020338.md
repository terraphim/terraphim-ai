## Task Description

Security fix (P2): `checkout.rs:108` constructs the target path as
`checkout_root.join(owner).join(repo)` without validating that `owner` or `repo`
contain no path-traversal components (`..`, absolute paths).

A forged task payload with `repository: "../../sensitive"` escapes `checkout_root`.

4. [S] Add 4 regression tests (traversal in owner, traversal in repo, absolute owner, empty)
