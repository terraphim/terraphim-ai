# DevOps Knowledge Graph

## Containerisation

Containerisation is the packaging of software code with all its dependencies into a single unit.

[synonyms::] container, docker, podman, containerisation

## Orchestration

Orchestration automates the deployment, scaling, and management of containerised applications.

[synonyms::] kubernetes, k8s, orchestration, helm

## Infrastructure as Code

Infrastructure as Code manages infrastructure through machine-readable definition files.

[synonyms::] terraform, ansible, pulumi, iac

## CI/CD

Continuous Integration and Continuous Delivery automate software release pipelines.

[synonyms::] ci, cd, pipeline, github-actions, gitlab-ci

## Observability

Observability measures system state through logs, metrics, and traces.

[synonyms::] monitoring, logging, tracing, prometheus, grafana

## Cloud Platforms

Cloud platforms provide on-demand computing resources over the internet.

[synonyms::] aws, azure, gcp, cloud, serverless

## Networking

Networking configures communication between services and hosts.

[synonyms::] ingress, egress, load-balancer, vpc, dns

## Secrets Management

Secrets management securely stores and distributes sensitive credentials.

[synonyms::] vault, secret, credential, certificate, tls

## Deployment Strategies

Deployment strategies control how new versions are rolled out to production.

[synonyms::] blue-green, canary, rolling-update, rollback

## DevSecOps

DevSecOps integrates security practices into the DevOps lifecycle.

[synonyms::] security, vulnerability, scan, compliance, sbom
