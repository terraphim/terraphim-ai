   - Settings.github_webhook_secret, github_token, firecracker_auth_token (lines 15-27)

## Remediation

Apply custom `fmt::Debug` redaction pattern (see LinearConfig in tinyclaw/config.rs for reference):

```rust
struct TelegramConfig {
    token: String,
}

impl Debug for TelegramConfig {
    fn fmt(&self, f: &mut Formatter<'_>) -> Result {
        f.debug_struct(\"TelegramConfig\")
            .field(\"token\", &\"***REDACTED***\")
            .finish()
    }
}
```
