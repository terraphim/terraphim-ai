Context Menu Actions

The extension provides the following context menu actions when you right-click on selected text:

  • Search in Terraphim AI: Search for the selected text using your configured server
  • Add Selection to LogSeq: Quick capture to LogSeq's daily notes
  • Add to Atomic Server: Add selected text to your Atomic Server instance
  • Concept Refactoring: Analyze concepts (coming soon)
