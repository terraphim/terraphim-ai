## 1. Critical Vulnerabilities (CVEs)

### CRITICAL-001: AWS-LC X.509 Name Constraints Bypass (RUSTSEC-2026-0044)
- **Crate:** aws-lc-sys v0.38.0
- **Severity:** HIGH
- **Date:** 2026-03-19
- **Impact:** X.509 certificate validation bypass via wildcard/unicode CN
- **Attack Vector:** Man-in-the-middle attacks possible against TLS connections
- **Solution:** Upgrade to >=0.39.0
- **Affected:** rustls, ureq, reqwest (entire networking stack)

### CRITICAL-002: AWS-LC CRL Distribution Point Scope Error (RUSTSEC-2026-0048)
- **Crate:** aws-lc-sys v0.38.0
- **Severity:** 7.4 (HIGH)
- **Date:** 2026-03-19
- **Impact:** Certificate revocation check bypass
- **Solution:** Upgrade to >=0.39.0

### CRITICAL-003: IDNA Punycode Validation Bypass (RUSTSEC-2024-0421)
- **Crate:** idna v0.4.0
- **Severity:** MEDIUM
- **Date:** 2024-12-09
- **Impact:** Domain name validation bypass for Punycode labels
- **Solution:** Upgrade to >=1.0.0
- **Affected:** trust-dns-proto/resolver in terraphim_rlm

### CRITICAL-004: rustls-webpki CRL Matching Logic Error (RUSTSEC-2026-0049)
- **Crate:** rustls-webpki v0.102.8
- **Severity:** MEDIUM
- **Date:** 2026-03-20
- **Impact:** Certificate revocation lists not properly validated
- **Solution:** Upgrade to >=0.103.10
- **Affected:** rustls, tungstenite, tokio-tungstenite (WebSocket connections)
