## API Reference Snippets

### QualityScore (New in v1.15.0)

```rust
use terraphim_types::{QualityScore, IndexedDocument};

// QualityScore tracks KLS dimensions:
// K (Krogstie) - Structural quality
// L (Lindland) - Content quality  
// S (Sindre) - Presentation quality

let score = QualityScore::default()
    .with_k(0.85)// Structure quality
    .with_l(0.92)// Content quality
    .with_s(0.78);// Presentation quality

let doc = IndexedDocument {
    id: "doc-1".into(),
    // ... fields
    quality_score: Some(score),
};```

### LayeredSearch Output (New in v1.15.0)

```rust
use terraphim_types::{SearchQuery, SearchLayer};

// Layer 1: Raw results (full documents)
// Layer 2: Summarized results (key sections)
// Layer 3: Condensed results (key terms only)

let query = SearchQuery {
    search_term: "async rust patterns".into(),
    layer: Some(SearchLayer::Layer2), // Summarized
    limit: Some(10),
    ..Default::default()
};
```
