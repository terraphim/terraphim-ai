# Quickstart

Fast paths for getting Terraphim AI running.

## Install (recommended)

```bash
curl -fsSL https://raw.githubusercontent.com/terraphim/terraphim-ai/main/scripts/install.sh | bash
```

## Run from source

```bash
cargo build --workspace
cargo run -p terraphim_server
```
