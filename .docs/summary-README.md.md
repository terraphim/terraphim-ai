# Summary: README.md

**Purpose:** Project overview and installation instructions.

**Key Details:**
- Terraphim AI: privacy-first, local AI assistant
- Multi-language packages:
  - Rust: `terraphim_agent` (crates.io)
  - Node.js: `@terraphim/autocomplete` (npm)
  - Python: `terraphim-automata` (PyPI)
- Installation methods:
  - Universal installer script (scripts/install.sh)
  - Homebrew: `brew tap terraphim/terraphim`
  - Cargo: `cargo install terraphim_agent`
- Key features: semantic knowledge graph search, smart text linking, offline-capable
- Resource usage: ~15MB RAM, ~13MB disk, <200ms operations
- Links to: QUICKSTART.md, Changelog, Cross-Platform Guide
- Community: Discord, Discourse
