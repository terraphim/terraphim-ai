Fast routing prioritizes speed and low latency above all else. Routes to Groq for 100+ tokens/sec inference - the fastest available option.

route:: groq, llama-3.3-70b-versatile

- Real-time applications demand lowest latency
- Interactive coding requires instant feedback

- High throughput batch processing needed

- Uses Groq's ultra-fast inference (100+ tokens/sec)

- Good quality with Llama 3.3 70B model

- Time-critical automation
