Navigation Waypoints

Core Capabilities

Each waypoint represents a fundamental capability of the Terraphim navigation system.

WP-001

Local-First Architecture

Your data never leaves your machine. All indexing, searching, and graph traversal happens entirely on-device. No cloud dependency, no data exfiltration, no telemetry.

WP-002

Knowledge Graph Engine

Build and traverse interconnected knowledge graphs with Aho-Corasick multi-pattern matching. Thousands of patterns matched simultaneously in a single pass.

WP-003

MCP Server Integration

Native Model Context Protocol server. Connect your AI assistant to local knowledge graphs. KG-boosted file search, token tracking, and role-based context enrichment.

WP-004

Rust/WASM Performance

Written in Rust, compiled to WebAssembly. Sub-millisecond search latency. Memory-safe, thread-safe, running at near-native speed in your browser or desktop.

WP-005

Hooks System

Pre and post tool-use hooks for AI coding assistants. Capture failed commands, inject knowledge graph context, and automate learning from development sessions.

WP-006

KG-Boosted File Search

File search augmented with knowledge graph context. Find files not just by name but by their semantic relationship to your domain concepts and role-specific vocabulary.
